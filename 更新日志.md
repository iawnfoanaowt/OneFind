# 更新日志

## v1.5.2026.10611

发布日期：`2026-04-16`

1、插件名称更新为OneFind，中文名：知寻，Github：https://github.com/iawnfoanaowt/OneFind

2、增加文件夹的接口，可以把ppt、word、excel、markdown、pdf放在文件夹内，也视为知识库来调用、分析。

3、增加Notion、Mendeley Reference Manager、Citavi7三个知识库的本地接入，不用授权即可调用分析使用，当前总共支持7个数据源

4、增加支持自定义所有知识库接口的路径，新增、删除、禁用、启动

5、增加支持调用Zotero、Obsidian中存储的ppt、word、excel等附件文件来做分析

6、增加更新功能，点击安装即可覆盖更新，不再需要重复输入路径、授权、deep模块下载

7、优化新、旧版本Endnote在无PDF时，可检索其条目来回答，以应对科研人员仅下载摘要等条目信息，未下载PDF的情况。

8、优化索引刷新的逻辑，支持自动重建索引，减少用户等待时间。

9、修复deep模块安装GPU驱动版本不一致，导致实质调用CPU的问题。

10、修复一些已知错误，以提高使用体验

11、建立公众号：OneFind，发布更新、教程等更多内容


强烈建议安装1.4及以下版本的同学，卸载当前版本，重新安装1.5版本

旧版卸载工具：`C:\Users\你的用户名\.codex\Function\kb_gateway\KnowledgeBaseGatewayUninstall.exe`


## v1.4.2026.10104

发布日期：`2026-04-11`

- 支持 EndNote 多库接入，安装时最多可配置 3 个 `.enl`
- 支持通过自然语言新增、替换、禁用或移除知识库路径
- 修复部分 Obsidian 索引状态检查问题
- 增强安装器对不同知识库路径组合的兼容性
- deep 模式安装项默认勾选，并按 GPU / CPU 条件分流
