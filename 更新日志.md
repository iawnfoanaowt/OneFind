# 更新日志

## v1.4.2026.10104
发布时间：2026-04-11

- 增加 EndNote 多库接入：安装时最多可配置 `3` 个 EndNote `.enl` 文库，默认一起检索。
- 可通过自然语言的命令，来新增、替换、禁用 Obsidian/Zotero/EndNote 路径，修改后刷新索引即可，无需卸载重装。
- 修复 Obsidian 索引状态检查在 OneDrive、缺失文件或不可访问 Markdown 文件上中断安装的问题。
- 安装器界面优化，在不同电脑上定向自适应。
- deep 模式安装选项默认保持勾选；检测到 NVIDIA GPU 时安装 GPU 适配依赖，未检测到 NVIDIA GPU 时，可安装 CPU 版 deep 支持。
- 跳过了v1.3 测试版本

## v1.2.2026.10009
发布时间：2026-04-10

- 支持两类 `EndNote` 数据结构：
  - `sdb/sdb.eni`
  - `rdb/tdb + PDF`
- 安装器改进了 `EndNote` 路径校验逻辑
- 修复了安装阶段部分旧脚本导致的报错问题


## v1.1
发布时间：2026-04-09

- 支持 `Obsidian`、`Zotero`、`EndNote` 三类本地知识源的统一接入


## v1.0
发布时间：2026-04-08

- 首次完成 `Knowledge Base Gateway` 基础版本
- 支持将 `Obsidian`、`Zotero` 本地知识库接入 `ChatGPT Codex`
- 建立本地索引、快速检索与 `deep` 深度分析的基本能力
- 支持论文、方法、综述、科研问答等典型科研场景
