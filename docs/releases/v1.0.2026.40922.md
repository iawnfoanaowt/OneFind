# Knowledge Base Gateway v1.0.2026.40922

发布日期：`2026-04-09`

## 版本摘要

这是首个面向发布整理的安装分发版本，包含图形化安装器、图形化卸载器与便携包三类交付物，并配套安装说明、版权声明、界面截图与校验信息，方便后续维护和版本迭代。

## 本版本功能范围

- 面向 Codex 的本地知识库检索网关。
- 支持 `Obsidian` 与 `Zotero` 双源检索，以及单源模式。
- 采用本地索引与检索融合策略，支持 Markdown、文献元数据、笔记、批注与全文缓存文本。
- 提供 `fast` / `balanced` / `deep` 三档模式。
- `deep` 模式支持语义检索与 rerank，并可在具备 NVIDIA GPU 时优先启用 GPU。
- 提供常用命令入口：`kb_ask`、`kb_search`、`kb_report`、`kb_index`、`kb_refresh`、`kb_service`。
- 安装器支持旧版本卸载、私有 Python 运行时准备、deep 依赖安装、自检与只读路径约束。
